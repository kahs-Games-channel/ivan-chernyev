## Запуск программы

1. Установите Python 3.11+ на Windows.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение от имени администратора:
   ```bash
   python app.py
   ```

### Что нужно рядом с `app.py`

- папка `bin` с `winws_zapret.exe` и бинарными шаблонами TLS/QUIC;
- папка `lists` со всеми `.txt`-списками;
- опционально `icon.ico`, `icon1.ico`, `icon2.ico`, `background.png`, `service.bat`.
